# GitHubprilingProject

## חוקי הצוות - Team Rules

### כללי עבודה בפרויקט

**חובה לקרוא ולעקוב אחר החוקים הבאים:**

1. **אסור להוסיף קוד או סקריפט ללא אישור מראש**
   - כל תוספת קוד חייבת לקבל אישור ממנהל הפרויקט
   - לא להעלות קבצים ללא בדיקה ואישור

2. **Claude Code פועל רק באישור**
   - Claude Code מורשה לפעול רק עם הרשאה מפורשת
   - לא להפעיל אוטומציות ללא תיאום

3. **פתיחת ענף (Branch) דורשת אישור**
   - אסור ליצור סניפים חדשים ללא אישור מראש
   - רק ענף MAIN מאושר לעבודה
   - יצירת סניפים מוגבלת (Restrict creations enabled)

4. **עדכונים בענף MAIN בלבד**
   - כל העדכונים והשינויים מתבצעים רק בענף הראשי MAIN
   - אין לעבוד בסניפים נוספים ללא אישור מיוחד

### נהלי בקרה

- המאגר מוגדר עם הגבלות גישה
- Claude Code הוגדר כ-bypass עם הרשאות מלאות
- הגנה על ענף MAIN: Block force pushes, Restrict deletions, Restrict creations

---

**לכל שאלה או בקשת אישור, יש לפנות למנהל הפרויקט.**
